# 通过 xk6-milvus 项目学习 Golang

欢迎来到这个实践导向的 Golang 学习教程！

## 教程理念

**学中做，做中学** —— 本教程不是枯燥的语法讲解，而是通过一个真实的开源项目 `xk6-milvus` 来学习 Golang。你将：

- 阅读真实的生产代码
- 理解每一行代码的含义
- 动手修改和扩展代码
- 通过实践巩固知识

## 项目简介

`xk6-milvus` 是一个 k6 负载测试扩展，用于测试 Milvus 向量数据库。它包含了 Golang 开发中的多种常见模式：

- 包管理和模块系统
- 接口和结构体设计
- 错误处理
- JSON 序列化
- 类型转换
- 测试编写

## 开始之前

### 推荐的学习方式

1. **克隆项目到本地**
   ```bash
   git clone https://github.com/mmga-lab/xk6-milvus.git
   cd xk6-milvus
   ```

2. **准备好 IDE**
   - 推荐使用 VS Code + Go 扩展
   - 或者 GoLand

3. **边读边做**
   - 每章都有练习，请务必动手完成
   - 不要只是阅读，要运行代码

4. **记录笔记**
   - 遇到不理解的地方记下来
   - 完成练习后写下心得

### 预备知识

- 基本的编程概念（变量、函数、循环）
- 命令行基础操作
- 任意一门编程语言的经验（JavaScript、Python 等皆可）
